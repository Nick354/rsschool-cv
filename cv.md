## Пример кода:

**Задача с сайта Codewars:**

Если дана строка цифр, то вы должны заменить любую цифру ниже 5 на '0', а любую цифру 5 и выше на '1'. Верните результирующую строку.

**Код:**
```
function fakeBin(x){
let str = ''
for (let i = 0; i < x.length; i++) {

if (x[i]>=5) {
str = str + "1"
} else {
str = str + "0"
}
} return str
}
```
